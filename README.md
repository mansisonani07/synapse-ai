## ⚡ The Solution

**Synapse-AI is the self-driving car for B2B workflows.**

You describe what you want in plain English. A swarm of **5 specialized AI agents** takes over — reading documentation, writing production code, testing in sandboxes, securing it, and deploying. End to end. Autonomously.

```
USER: "Connect my Stripe to Slack when payment succeeds"
         ↓
    [AI SWARM]
         ↓
RESULT: Production-ready Python code in 30-60 seconds
```

## 🐝 The 5 AI Agents

| Agent | Emoji | Role | What It Does |
|-------|-------|------|--------------|
| **Orchestrator** | 🧠 | The Manager | Reads your prompt and delegates work to all other agents |
| **DocReader** | 📖 | The Researcher | Autonomously fetches and reads external API documentation |
| **CodeWriter** | ✍️ | The Developer | Generates production-ready Python or Node.js code |
| **Tester** | 🧪 | The QA Engineer | Runs code in a secure sandbox to catch every bug |
| **Validator** | ✅ | The Security Inspector | Audits for exposed API keys and security issues |

## 🚀 Local Setup

### Prerequisites
- Node.js 18+ ([Download](https://nodejs.org))
- Python 3.11+ ([Download](https://python.org))
- Groq API key ([Get free key](https://console.groq.com))
- Git ([Download](https://git-scm.com))

---

### Step 1 — Clone the Repository

```bash
git clone https://github.com/mansisonani07/synapse-ai.git
cd synapse-ai
```

---

### Step 2 — Setup Backend

```bash
# Navigate to backend
cd backend

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (Mac/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
cp .env.example .env
```

Edit `.env` file and add your Groq API key:
```env
GROQ_API_KEY=gsk_your_actual_key_here
GROQ_MODEL=llama-3.3-70b-versatile
ENVIRONMENT=development
FRONTEND_URL=http://localhost:5173
```

Start the backend:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

✅ Backend runs at: **http://localhost:8000**
📚 API Docs at: **http://localhost:8000/docs**

---

### Step 3 — Setup Frontend

Open a NEW terminal:

```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Start dev server
npm run dev
```

✅ Frontend runs at: **http://localhost:5173**

---

### Step 4 — Test Everything

1. Open: http://localhost:5173
2. Scroll to "Launch Your First Swarm" form
3. Fill in:
   - Integration Name: `Stripe to Slack Test`
   - Tech Stack: `Python`
   - Source API: `Stripe` (https://stripe.com/docs/api)
   - Target API: `Slack` (https://api.slack.com/web)
4. Click **🚀 LAUNCH MULTI-AGENT SWARM**
5. Watch the AI agents work their magic!

## 🌍 Deployment

### Frontend (Vercel)

1. Push code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import repository
4. Set Root Directory: `frontend`
5. Add Environment Variable:
   ```
   VITE_API_URL=https://your-backend.onrender.com/api/v1
   ```
6. Click Deploy

### Backend (Render)

1. Go to [render.com](https://render.com)
2. Create New Web Service
3. Connect GitHub repo
4. Set Root Directory: `backend`
5. Build Command: `pip install -r requirements.txt`
6. Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
7. Add Environment Variables:
   ```
   GROQ_API_KEY=your_key
   GROQ_MODEL=llama-3.3-70b-versatile
   ENVIRONMENT=production
   FRONTEND_URL=https://your-app.vercel.app
   ```
8. Click Deploy
